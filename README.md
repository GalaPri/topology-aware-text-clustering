# Clustering with Local Topology Classifier

Реализация метода кластеризации текстовых embeddings, основанного на анализе **локальной топологии пространства данных**.

Основная идея метода — сначала обучить классификатор, который определяет, является ли точка **внутренней точкой кластера или его границей**, используя **топологические характеристики окрестности**. После этого кластеры восстанавливаются на основе внутренних точек и соседних связей.

Метод использует инструменты **Topological Data Analysis (TDA)** и **persistent homology**.
Для векторизации тексте использована модель sentence-transformers/paraphrase-multilingual-mpnet-base-v2
---

# Идея метода

При векторизации текстов с помощью современных моделей (например SentenceTransformer) документы представляются точками в высокоразмерном пространстве.

В таких пространствах часто возникает проблема:

- документы разных тематических областей могут находиться близко
- границы кластеров оказываются размытыми
- классические алгоритмы кластеризации (например DBSCAN) работают нестабильно

В данной работе предлагается следующий подход:

1. Рассмотреть локальную окрестность точки
2. Построить для неё Vietoris–Rips комплекс
3. Вычислить persistent homology
4. Использовать топологические признаки для классификации точки как:
   - внутренней точки кластера
   - граничной точки

---

# Используемые признаки

Для каждой точки вычисляются топологические характеристики её окрестности:

**Betti numbers**
H0_num
H1_num

**Сумма времен жизни гомологических компонент**
H0_persistence_sum
H1_persistence_sum


**Персистентная энтропия Шеннона**
H0_entropy
H1_entropy


Эти признаки используются как входные данные для классификатора.

---

# Классификатор

Для классификации внутренних и граничных точек используется **Random Forest**.

Модель обучается на размеченном наборе данных, где границы кластеров определяются с помощью ближайших соседей.
---

# Реконструкция кластеров

После классификации точек кластеры восстанавливаются следующим образом:

1. внутренние точки используются как ядра кластеров
2. граничные точки присваиваются кластеру большинства соседей
3. процесс повторяется до стабилизации
